# Référence — Outils SEO à recommander

Fichier à consulter pour suggérer des outils adaptés à l'utilisateur selon son besoin.

## Outils Google (gratuits, fondamentaux)

### Google Search Console
- **Rôle** : monitoring SEO pur — indexation, requêtes qui amènent du trafic, CTR, problèmes techniques
- **Quand le recommander** : à chaque fois qu'on parle de suivi de performance SEO
- **Prérequis** : propriétaire du site ou accès équivalent
- **À mentionner systématiquement** quand l'utilisateur parle de mesure de performance

### Google Analytics (GA4)
- **Rôle** : analytics global (toutes sources : SEO, SEA, social, direct, email…)
- **Limite** : interface jugée peu ergonomique depuis le passage à GA4
- **Alternatives recommandables** : Matomo (open-source), Plausible (RGPD-friendly), Fathom, Piwik

### Google Keyword Planner
- **Rôle** : volumes de recherche, suggestions de mots-clés
- **Accès** : gratuit via un compte Google Ads
- **Limite** : fourchettes de volume larges si aucune campagne active
- **Quand le recommander** : étape de validation des mots-clés

### Google Trends
- **Rôle** : mesure de l'évolution d'intérêt pour une requête dans le temps
- **Utilité** : saisonnalité, tendances émergentes vs déclinantes, comparaisons géographiques

### Google Alertes
- **Rôle** : veille automatisée par mail sur un mot-clé
- **Utilité** : trouver des idées de sujets, surveiller la concurrence, suivre l'e-réputation

## Outils de keyword research & audit

### Semrush (payant, référence)
- **Fonctionnalités** : KW research, audit technique, backlinks, suivi de positions, analyse concurrence
- **Public** : agences, pros du SEO
- **Prix** : ~120-450 €/mois
- **Quand le recommander** : pour un professionnel ou une entreprise qui investit sérieusement dans le SEO

### Ahrefs (payant)
- **Fonctionnalités** : similaire à Semrush, réputé pour sa base de backlinks
- **Prix** : équivalent à Semrush
- **Quand le recommander** : quand on veut analyser en détail le netlinking

### Haloscan (français, excellent)
- **Fonctionnalités** : KW research spécialisé marché FR, SERP similaires, questions
- **Intérêt** : outil français de qualité, souvent plus précis sur le marché hexagonal
- **Quand le recommander** : pour des projets ciblant la France

### Ubersuggest (Neil Patel, freemium)
- **Fonctionnalités** : KW research de base, idées de contenu
- **Prix** : version gratuite limitée, pro autour de 30-50 €/mois (ou licence à vie)
- **Quand le recommander** : pour démarrer sans gros budget

### Semrank.io
- **Fonctionnalités** : analyse de l'intention de recherche d'un mot-clé
- **Quand le recommander** : quand on hésite sur l'intention d'une requête

### Answerthepublic
- **Fonctionnalités** : génère un nuage de questions autour d'un mot-clé
- **Utilité** : idéation de contenu, optimisation pour la recherche vocale et les FAQ
- **Prix** : gratuit avec limitations, puis ~99 $/mois

## Outils d'audit technique

### Screaming Frog SEO Spider
- **Rôle** : crawler un site complet pour détecter erreurs 404, redirections, balises manquantes, duplicate content, structure du maillage
- **Prix** : gratuit jusqu'à 500 URLs, payant ensuite (~200 €/an)
- **Quand le recommander** : audit technique, migration de site, cartographie du maillage interne

### PageSpeed Insights (Google)
- **Rôle** : mesure la vitesse et les Core Web Vitals
- **Gratuit**

### GTmetrix
- **Rôle** : alternative à PageSpeed Insights, avec historique
- **Freemium**

### Lighthouse (intégré à Chrome)
- **Rôle** : audit complet (performance, SEO, accessibilité, bonnes pratiques)
- **Gratuit, intégré au navigateur**

## Outils de création & optimisation de contenu

### Yoast SEO (WordPress)
- **Rôle** : extension WordPress qui gère les balises title, meta description, sitemaps
- **Prix** : version gratuite suffisante pour la majorité des sites, version premium ~99 $/an

### Rank Math (WordPress)
- **Rôle** : alternative à Yoast, souvent considérée comme plus complète en gratuit
- **Prix** : gratuit avec version Pro

### YourTextGuru / SEOQuantum / Thot SEO
- **Rôle** : analyse du champ lexical et des mots-clés à intégrer dans un contenu pour être concurrentiel
- **Public** : rédacteurs web / content managers
- **Prix** : ~30-100 €/mois

### Surfer SEO
- **Rôle** : optimisation de contenu basée sur l'analyse des concurrents en Top 10
- **Prix** : ~70-250 $/mois

## Outils d'analyse comportementale

### Hotjar
- **Fonctionnalités** : heatmaps, enregistrements de navigation, feedback utilisateur
- **Utilité SEO** : comprendre pourquoi les visiteurs quittent une page (signal UX lu par Google)
- **Prix** : freemium, puis ~30-170 $/mois

### Microsoft Clarity
- **Fonctionnalités** : similaire à Hotjar
- **Prix** : **entièrement gratuit**
- **À mentionner quand** : l'utilisateur veut du Hotjar-like sans budget

## Outils pour découvrir la stack d'un concurrent

### Wappalyzer
- **Rôle** : identifier les technologies utilisées par un site (CMS, framework, analytics, plugins)
- **Utilité** : benchmark concurrentiel, choix technologique
- **Prix** : extension gratuite

### BuiltWith
- **Rôle** : similaire à Wappalyzer, plus poussé
- **Freemium**

## Outils de structured data / données structurées

### Outil d'aide au balisage Google
URL : `https://www.google.com/webmasters/markup-helper/`
- **Rôle** : générer le balisage schema.org pour des types précis (Article, Produit, Événement, FAQ…)

### Test des résultats enrichis
URL : `https://search.google.com/test/rich-results`
- **Rôle** : vérifier que les données structurées sont bien interprétées par Google

### Schema.org
URL : `https://schema.org`
- **Rôle** : référence officielle de tous les types de balisages disponibles

## Outils de raccourcissement d'URL

### Bitly
- **Utilité** : partager des URLs propres, tracking de clics
- **Freemium**

## Création de persona

### HubSpot Make My Persona
URL : `https://www.hubspot.fr/make-my-persona`
- **Rôle** : générateur gratuit de personas marketing
- **Utilité** : structurer la cible avant de rédiger

## Gestion de noms de domaine

### Gandi, OVH, GoDaddy
- **Rôle** : enregistrer et gérer des noms de domaine
- **Quand mentionner** : conseils sur le choix du NDD, sécurisation multi-extensions

## Stratégie de recommandation d'outils

Quand l'utilisateur demande « quel outil pour X », proposer **2-3 options** et pas une liste interminable :
- Une option **gratuite** (ou freemium généreux)
- Une option **pro/référence** (payante)
- Éventuellement une option **française** si pertinente

Toujours donner une recommandation **contextualisée** : un indépendant n'a pas besoin de Semrush à 450 €/mois. Une agence SEO oui.
