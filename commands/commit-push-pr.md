## Contexte

- Statut git actuel : !`git status`
- Diff git (staged et unstaged) : !`git diff HEAD`
- Branche actuelle : !`git branch --show-current`

## Ta tâche

En te basant sur les changements ci-dessus :

1. Créer une nouvelle branche si tu es sur main/master
2. Créer un seul commit avec un message conventionnel (type(scope): description)
3. Pusher la branche vers origin
4. Créer une pull request avec `gh pr create`

Effectue toutes ces opérations. Ne génère pas de texte supplémentaire.
