## Contexte

- Statut git actuel : !`git status`
- Diff git (staged et unstaged) : !`git diff HEAD`
- Branche actuelle : !`git branch --show-current`
- Commits récents : !`git log --oneline -10`

## Ta tâche

En te basant sur les changements ci-dessus, crée un seul commit git.

Stage les fichiers modifiés et crée le commit avec un message clair et conventionnel (format: type(scope): description).
Ne fais rien d'autre. Ne génère pas de texte supplémentaire.
